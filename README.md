# Kamel-Library-System
本系統為設計給海大圖書館使用之線上圖書系統

##    系統建置
### 軟體需求
##### 1. Xampp v3.3.0(將佔用80,443,3306 port)
##### 2. Python 3.10.2 
##### 包含以下模組
        1.requests(2.27.1)
        2.beautifulsoup4(4.10.0)
        3.fake-useragent(0.1.11)
##### 3. Windows作業系統

##### 備註:若需使用寄信功能，建議將防火牆關閉，以免寄信失敗
---
